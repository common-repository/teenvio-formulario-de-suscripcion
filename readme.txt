Word Press plugin that generate a widget to show a subscribe form 

== Description ==

Simple WordPress plugin to generate a form so that your e-mail can be included in the list of newletters

== Installation ==

1. Upload `tesu-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin in the > 'Settings' > 'Teenvio'
